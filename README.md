# AtliQO-Bank-Credit-Card-Analysis

Project Goal: To find an untapped market for a new credit card.

Data Cleaning: Handled outliers in Age (18-64) and fixed Income/Transaction errors.

Key Discovery: The 18-25 age group is a huge opportunity (26% customers).

Statistical Proof: Used Z-Test (Z-score 2.76, p-value 0.0) to prove the campaign's success.
